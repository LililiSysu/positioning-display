本程序实现了从网络侧接收设备位置信息，并在网页地图显示设备位置的功能。
用户操作界面基于python的 tkinter 库实现，网页地图基于百度地图api实现。

文件夹说明

testing：测试所用的数据，以此发现在开发过程中的问题。在最终使用本程序时，无需使用testing文件夹中的内容

total：此文件夹存放实现前述功能的所有程序，在其下的子文件夹中，total文件夹包含的exe文件是本程序的入口，其它文件为辅助文件，实现网页地图的显示和用户数据的记录。子文件夹total_store包含程序与设备在信息交换过程中所用到的测试程序，数据格式的规定，数据存储方式等。

代码：此文件夹包含网页地图显示代码，和用户界面代码。
